# personDB

## 디렉토리 설명
- backend : API 서비스를 위한 SpringBoot 기반 백엔드
- fo : 일반 사용자를 위한 ReactJS 기반 프론트엔드
- bo : Admin을 위한 NextJS 기반 프론트엔드

### NextJS (fo) 실행 방법
```
$ cd fo
$ npm install 
$ npm run dev
```
### ReactJS (bo) 실행 방법
```
$ cd bo
$ npm install 
$ npm start
```
